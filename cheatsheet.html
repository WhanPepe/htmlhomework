Paragraphs

Paragraphs are denoted by the p tag. To create dummy text input "lorem" plus the number of words you require then press tab.

The output of lorem8:

Lorem ipsum dolor sit amet consectetur adipisicing elit.

Paragraphs are block elements. They are automatically shown on a new line. They also have a margin

Inline .vs Block Level Elements

  1. Inline elements:
    1. Do not start on new line.
    2. Take only the necessary width.
    3. Examples: span, img, a, strong, em
  2. Block Elements:
    1. Start on new line.
    2. Take full width available.
    3. Examples: div, headings, paragraphs, form

Images

Insert images using the starter img. Use the src attribute for the image's file path. You can also edit the image size using width and height attributes. Note that images are inline.

Images can have links wrapped around them using a.
